IoT 硬件

GPS 宠物定位器

基于 STM32F103 + FreeRTOS 的宠物追踪设备。通过 AT6558R GPS 模块定位、DS3553 计步，每 10 秒采集一次数据，经 QS100 NB-IoT 上报，LoRa(LLCC68) 备用链路。JSON 格式包含经纬度、步数、时间戳。

解决什么让宠物定位、计步与云端上报形成稳定的数据闭环。 核心技术任务调度、串口解析、GPS/NB-IoT/LoRa 通信链路。 成果亮点10 秒周期采集，NB-IoT 主链路上报，LoRa 作为备用通信链路。
STM32 FreeRTOS GPS NB-IoT LoRa CMake

移动应用

Daily — 今日打卡了吗？

一款跨平台学习打卡应用（v2.0.1），帮助用户养成每日学习好习惯。基于 uni-app (Vue 3) + uniCloud (阿里云) 构建，支持 Android、iOS、微信小程序及 H5 多端运行。功能包括一键打卡与完成率统计、月历视图与滑动切换、任意日期日程管理与提醒，以及通过一言 API 自动获取每日励志名言，具备离线备用容错能力。根据星期自动切换主题配色，采用卡片式设计与流畅动画。

解决什么把每日打卡、日历复盘和学习提醒放进一个轻量应用。 核心技术uni-app 多端框架、Vue 3 状态管理、uniCloud 云函数与数据库。 成果亮点跨端运行、离线备用、按星期自动切换主题与名言推送。
uni-app Vue 3 uniCloud SCSS 跨平台 HBuilderX

个人主页网站

一个用于展示个人技能、项目经历与联系方式的响应式作品集网站。页面基于原生 HTML/CSS/JavaScript 构建，包含柔和的视觉风格、气泡背景、入场动画、滚动 reveal、返回顶部按钮与 GitHub 项目跳转。开发过程中使用 AI 工具协助页面结构梳理、交互调试、文案优化与细节修复。

解决什么把个人介绍、工程能力和项目入口收束到一个可发布页面。 核心技术原生 HTML/CSS/JavaScript、Canvas 动效、IntersectionObserver。 成果亮点首屏作品预览、玻璃拟态视觉、响应式布局与 AI 协作迭代。
