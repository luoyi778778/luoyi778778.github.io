## 核心特性

- **旋转开页动画** — 页面加载时从 -90° 旋转展开，配合 LOGO 遮罩淡出
- **气泡粒子背景** — Canvas 绘制 18 个漂浮气泡，淡蓝色调径向渐变
- **滚动触发动画** — IntersectionObserver 监听 4 种方向的入场动画
- **玻璃拟态卡片** — backdrop-filter 模糊 + 半透明背景 + 微边框
- **CSS 变量系统** — 颜色 / 阴影 / 圆角全部可配置，便于主题扩展
